A numeral interpreter is the pair of an interpreter for **decimal** numbers in terms and an optional interpreter in pattern, if non integer or negative numbers are not supported, the interpreter must fail with an appropriate error message

type notation_location = (Names.DirPath.t * Names.DirPath.t) * string

type rawnum = Constrexpr.sign * Constrexpr.raw_numeral

The unique id string below will be used to refer to a particular registered interpreter/uninterpreter of numeral or string notation. Using the same uid for different (un)interpreters will fail. If at most one interpretation of prim token is used per scope, then the scope name could be used as unique id.

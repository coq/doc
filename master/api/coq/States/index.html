Module States

States of the system
  • Rollback

In that module, we provide functions to get and set the state of the whole system. Internally, it is done by freezing the states of both Lib and Summary. We provide functions to write and restore state to and from a given file.

val lib_of_state : state -> Lib.frozen
val replace_summary : state -> Summary.frozen -> state
val replace_lib : state -> Lib.frozen -> state

register h registers h as a quickfix generator. When an error is generated the toplevel can check if there is quick fix for it.

Quickfix generators signal that they don't deal with some exception by returning . Raising any other exception is forbidden.

Exceptions that are considered anomalies should not be handled by Quickfix generators.
